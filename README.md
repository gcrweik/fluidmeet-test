## Installation Guide

Please follow these instructions:

- Create a database called "fluidmeet_db".
- Import the sql file found in the main directory, called "fluidmeet_db.sql".
- Fix the .env file like if there's any changes you have to make (I don't think there will be if you're running the project on a local machine).
- Open a terminal and change directory to the project's main directory.
- Type this command "php artisan serve".
- The project is running.


## Main Routes

For the front end:
- http://127.0.0.1:8000/en

For the admin panel:
- http://127.0.0.1:8000/admin
